# conference-schedule-releases

Signed release APK downloads for **Conference Schedule** — an Android app for browsing and
planning a conference schedule. The current build ships the SIGGRAPH 2026 schedule. Not on the
Play Store yet, so this repo exists purely to host a directly-downloadable, publicly-accessible
APK in the meantime.

## Install

**Easiest:** open the [download page](https://wnschnapp.github.io/conference-schedule-releases/)
on your Android phone — it always serves the latest release.

Or by hand:

1. On your Android phone, download the latest APK from the [Releases page](https://github.com/wnschnapp/conference-schedule-releases/releases).
2. Open the downloaded file. Android will prompt you to allow "install unknown apps" for
   whichever app you downloaded it with (browser, Files, etc.) — this is expected for any app
   installed outside the Play Store, and only needs to be allowed once.
3. Install and open.

**Versionless direct link** (always the latest release, so it never needs updating):
https://github.com/wnschnapp/conference-schedule-releases/releases/latest/download/androidApp-release.apk

## What is this?

The actual app source lives in a separate, private repository. This repo only ever contains
compiled release builds — no source code, no bundled conference data.
